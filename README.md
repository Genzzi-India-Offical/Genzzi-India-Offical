## What is Genzzi?

**Genzzi** is not just an OAuth provider. It's a complete, interconnected ecosystem — code editor, quiz platform, LMS, mail, and HRM — all unified under **one Indian login**.

> One account. Six products. Zero friction. No foreign servers. No USD bills.

India has 900 million people online. Until now, zero Indian OAuth providers existed. Every "Sign in with Google" button sent your users' identity to a US company. Every Auth0 plan billed in USD. **Genzzi fixes that.**

## 📦 For Developers — Add Genzzi OAuth to Your App

Same flow as Google OAuth. Your users sign in with Genzzi. You get their details via token. Three packages, any JS stack.

### Install

```bash
# SPA authentication
npm install @genzzi/oauth

# Full stack — frontend
npm install @genzzi/oauth-client

# Full stack — backend
npm install @genzzi/oauth-server
```

| Package | Use Case |
|---|---|
| `@genzzi/oauth` | SPA apps |
| `@genzzi/oauth-client` | Full stack frontend |
| `@genzzi/oauth-server` | Full stack backend |

That's it. Same as Google OAuth. Indian provider. INR pricing.
